# LEN APP

> [cite_start]Tu privacidad y la seguridad de tus datos son nuestra máxima prioridad. 

[cite_start]LEN APP es una aplicación móvil diseñada para ayudar a los estudiantes a organizar sus planes de estudio, calificaciones, enlaces y recordatorios de manera completamente privada y segura.  

## 🚀 Características

* [cite_start]Gestión completa de planes de estudio y asignaturas. 
* [cite_start]Registro y categorización de calificaciones. 
* [cite_start]Almacenamiento seguro de enlaces útiles. 
* [cite_start]Sistema de recordatorios y alertas. 

## 🔒 Privacidad y Arquitectura de Datos

[cite_start]La aplicación está construida bajo una estricta política de privacidad "Local-First" (100% offline): 

* [cite_start]**Almacenamiento Local:** Absolutamente toda la información ingresada se almacena en una base de datos interna (SQLite) en el propio dispositivo del usuario.  [cite_start]Solo la aplicación instalada tiene acceso a estos datos. 
* [cite_start]**Sin Servidores Externos:** LEN APP no cuenta con servidores en la nube.  [cite_start]No recopila, transmite, comparte ni vende información a terceros, empresas de analítica ni plataformas publicitarias. 
* [cite_start]**Control del Usuario:** El usuario tiene el control total sobre sus datos.  [cite_start]Los registros individuales pueden ser eliminados en cualquier momento desde la interfaz. [cite: 7] [cite_start]Al desinstalar la aplicación o borrar los datos desde los ajustes del sistema operativo, toda la información se elimina de forma permanente e irrecuperable. [cite: 8]
* [cite_start]**Permisos Mínimos:** La app solicita únicamente el permiso de notificaciones para programar alertas de la herramienta Recordatorios.  [cite_start]No se solicita acceso a la ubicación, contactos, cámara, micrófono ni archivos externos.
